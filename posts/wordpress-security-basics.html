1. Change the Default admin Username

Using admin as your WordPress username is one of the most common ways hackers try to brute-force their way in. Change it to something unique and hard to guess. To do this safely, create a new admin user with a strong username and password, then delete the old admin account to close this easy attack vector.

2. Use a Trusted Security Plugin

Security plugins like Wordfence or iThemes Security add a powerful layer of protection. They can block brute force login attempts, scan for malware, monitor file changes, and notify you about vulnerabilities. Theyâ€™re easy to install and configureâ€”even if youâ€™re new to WordPress.

3. Keep WordPress, Themes, and Plugins Updated

Outdated software is the most common cause of WordPress site hacks. Developers regularly release patches that fix security flaws, so keeping your core WordPress files, themes, and plugins up to date is critical. Consider enabling automatic updates or check for updates at least once a week to stay ahead of threats.

4. Disable File Editing in the Dashboard

By default, WordPress lets admins edit plugin and theme files directly through the dashboard. This is convenient but risky if an attacker gains access. To disable this, add the following line to your wp-config.php file:

define('DISALLOW_FILE_EDIT', true);

This simple step closes off a critical attack surface.

5. Use Strong Passwords and Enable Two-Factor Authentication (2FA)

Passwords are your first line of defenseâ€”make them strong and unique. Password managers like Bitwarden can help generate and store complex passwords easily.

For even stronger security, enable 2FA on your admin accounts using apps like Authy or Google Authenticator. This means even if your password is compromised, attackers need your phone to get in.

Bonus: Backup Your Site Regularly

Sometimes, despite best efforts, things go wrong. Regular backups ensure you can restore your site quickly after an attack or error. Most hosts offer automatic backups, but you can also use plugins like UpdraftPlus for hassle-free backups to cloud storage.
